CWD3
====

## Development

Install Node.js:

```
https://nodejs.org
```

Clone project:

```
git clone git@github.com:gvcm/CWD3.git
```

```
cd CWD3
```

Install dependencies:

```
sudo npm install -g gulp
npm install
```

Run application:

```
gulp
```

## Production

### Deploying to Heroku

```
$ heroku create
$ git push heroku master
$ heroku open
```
